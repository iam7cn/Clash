- ClashForAndroid：Clash 的 Android 图形客户端，[Github](https://github.com/Kr328/ClashForAndroid)
- **Clash for Windows：Clash 的 Windows/macOS 图形客户端，[Github](https://github.com/Fndroid/clash_for_windows_pkg)**
